A centralized project management and client-decision platform for architecture firms: Decision Log, timeline, contextual messaging, and audit-ready approvals.

## Setup

- **Install:** `npm install` (use `npm install --include=dev` if `NODE_ENV=production` so devDependencies like Vite and TypeScript are installed)
- **Build:** `npm run build`
- **Lint:** `npm run lint`

## Tech stack

- React 18, React Router 6, TypeScript, Vite
- Tailwind CSS v3, Radix UI, Sonner, Recharts
- TanStack React Query, React Hook Form, Zod
